# MinerAI (Minerai)

MinerAI (Minerai) is a geospatial AI platform for satellite intelligence, dataset generation, and real-time monitoring — currently piloting **illegal mining detection**.

- Website: https://minerai.space
- Pilot map: https://minerai.space/map
- Built by: Xuremi

## What you can do with MinerAI
- Draw polygons on the map to define areas of interest
- Export training datasets (images + masks)
- Train segmentation models externally
- Run inference and visualize scan results

## Repositories
- `minerai-examples` — minimal inference scripts + runnable demos
- `minerai-sample-data` — tiny dataset (50 images + 50 masks)

## Status
Pilot phase (illegal mining). More solutions will be released progressively.
